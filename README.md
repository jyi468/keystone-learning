# Setup
`mkdir keystone-learning`
`cd keystone-learning`
`yarn init`
`yarn add @keystone-6/core`

You must create a `keystone.ts` file after.
`// keystone.ts`
`export default {};`

Add typescript as a dependency
`yarn add typescript`

Run with `yarn keystone dev`
